# Optical glass material system

The GitHub profile SVGs use one deterministic, self-contained optical material generated by `scripts/generate_optical_glass_assets.py`.

The SVG canvas remains transparent. Each pane is formed from restrained native SVG layers: two quiet shadows, a colour-neutral transmission gradient, a feathered pearl side-depth layer, broad internal illumination, paired refractive edges and isolated asymmetric specular highlights. Foreground text and icons are emitted after the material and are never blurred.

No pane uses raster backgrounds, embedded images, JavaScript, external CSS, `foreignObject` or backdrop filtering. Compact information cards reuse the same `chipGlass` and `chipEdge` definitions.

Regenerate and validate with:

```bash
python scripts/generate_optical_glass_assets.py
python scripts/validate_optical_glass.py
```

For rendered alpha validation, render each target SVG to a same-named PNG and pass its directory:

```bash
python scripts/validate_optical_glass.py --rendered-dir path/to/rendered
```
