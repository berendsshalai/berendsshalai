## Profile

I build practical systems that convert fragmented administrative work into structured, validated and auditable workflows.

My foundation is business operations: payroll hours, attendance exceptions, workforce planning, financial reconciliation, recruitment intake, document capture and recurring management reporting. My technical work turns those processes into reusable software through Python, Excel/VBA, TypeScript, React, Java, SQL and Model Context Protocol integrations.

```text
observe the workflow
        |
extract the rules
        |
validate the data
        |
automate repeatable decisions
        |
surface exceptions for human review
        |
document and open-source the reusable system
```

## Selected repositories

### [Onboarding Pilot AI](https://github.com/berendsshalai/onboarding-pilot-ai)

A privacy-conscious, configuration-driven Python workflow framework with validation, templating, recipient routing, persistent state, duplicate prevention, transactional delivery controls and synthetic demonstration data.

### [System to Portfolio](https://github.com/berendsshalai/berendsshalai-project-systemtoportfolio)

A technical case study for converting operational systems into inspectable public evidence through architecture documentation, secure publication, portfolio presentation and reproducible engineering records.

### [Attendance Reconciliation Engine](https://github.com/berendsshalai/attendance-reconciliation-engine)

A clean-room Python engine that reconciles planned shifts, neutral coverage requirements and clock events into auditable exception reports with tests, CI, documentation and GitHub Pages.

### [Sha-Lai Berends Systems Lab](https://github.com/berendsshalai/berendsshalai-systems-lab)

A control repository for turning private operational learning into sanitized, open-source project systems with publication gates, portfolio documentation and reusable explanation patterns.
